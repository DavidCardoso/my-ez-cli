Use specific, project-contextual names for Claude Code skills. Avoid generic names.

**Why:** User rejected `add-tool` as too generic; renamed to `add-mec-tool` to make context clear.

**How to apply:** When creating a new skill, include the project name or domain (e.g., `add-mec-tool`, `memory-migration`, `claudemd-streamline`) rather than short generic names like `add-tool`, `fix-bug`, `update`.
