# mec-dashboard

Local web dashboard for [My Ez CLI](../../README.md). Visualises all tool session logs and AI analysis results.

## Stack

- **Backend**: Python 3.12, FastAPI, uvicorn, watchfiles
- **Frontend**: Vue 3, Vite, PrimeVue 4 (Aura dark theme), Chart.js, marked.js, highlight.js
- **Distribution**: Single Docker container (multi-stage build)

## Usage

```shell
mec dashboard start    # Start (API + UI on http://localhost:4242)
mec dashboard stop     # Stop
mec dashboard restart  # Restart
mec dashboard status   # Show status and URL
mec dashboard open     # Open in default browser
```

## Pages

| Page | URL | Description |
|------|-----|-------------|
| Home | `/` | Stat cards + activity charts (sessions by tool, exit codes, last 7 days) |
| Sessions | `/sessions` | Session list — search and filter by tool, AI status, exit code |
| Session Detail | `/sessions/<id>` | Log output + AI analysis with markdown rendering and `claude --resume` hint |
| Tools | `/tools` | Read-only registry of all mec tool wrappers |

```
├── frontend/
│   ├── src/
│   │   ├── main.js          # Vue app entry, PrimeVue config
│   │   ├── App.vue          # Root component (NavBar + RouterView)
│   │   ├── router/          # Vue Router (history mode, 4 routes)
│   │   ├── composables/     # useWebSocket.js, useSessions.js
│   │   ├── components/      # NavBar, WsStatus, SessionTable, AiOutput, LogOutput
│   │   └── pages/           # HomePage, SessionsPage, SessionDetailPage, ToolsPage

```

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/api/sessions` | List sessions (newest first, limit 50) |
| `GET` | `/api/sessions/:id` | Session detail including log output and AI analysis |
| `GET` | `/api/stats` | Aggregate stats (totals, exit code distribution, AI rate, last 7 days) |
| `GET` | `/api/tools` | Tool registry (27 tools) |
| `WS` | `/ws` | Live updates broadcast when new sessions are written |

## Development

### Run tests (Python)

```shell
cd docker/dashboard && ./build   # build the image first
docker run --rm davidcardoso/my-ez-cli:dashboard-latest python -m pytest tests/ -v
```

### Run tests (Frontend)

```shell
cd services/dashboard/frontend
npm install
npm test          # Vitest unit tests
npm run test:e2e  # Playwright e2e tests
```

### Frontend dev server

```shell
mec dashboard start   # start API container on :4242

# in services/dashboard/frontend/:
MEC_BIND_PORTS="5173:5173" npm run dev -- --host 0.0.0.0 --port 5173
# open http://localhost:5173
```

### Build Docker image

```shell
cd docker/dashboard && ./build
mec dashboard restart
```
